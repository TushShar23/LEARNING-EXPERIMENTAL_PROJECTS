## ABOUT

```
This project is SIMPLE AUTHENTICATION APP.
> user and admin can signup.
> user and admin can sigin.
> After authentication server sends token as response.
```

### Technology/Libraries used

```
1 ExpressJS
2 MongoDB & mongoose
3 Bcrypt for encoding/hashing the password 
4 Zod for input validation
```

### Installation

```
1 First clone this project in your local machine
2 run "npm install" command to install all the dependencies that have been used in the project.
```
